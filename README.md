# tableau-googleplaystore
Project Overview: Google Play Store App Analysis
This project analyzes a dataset of over 10,000 applications from the Google Play Store to identify trends in ratings, reviews, pricing models, and content distribution.


the basis level analysis and key insights-

Rating: Average user rating-
Reviews: Number of user reviews- 
Installs: Number of downloads-
Type: Free or Paid-
Price: Cost of the app- 
Content Rating: Target audience- 
Last Updated: Date of the most recent update.-
